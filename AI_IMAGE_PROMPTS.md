# AI Image Generation Prompts for Portfolio

Use these prompts with **Midjourney**, **DALL-E 3**, or **Stable Diffusion** to generate portfolio images.

---

## General Style Guide

Add these to the end of any prompt for consistency:
```
--style: clean minimalist SaaS UI, modern design system, subtle shadows,
professional product design, Dribbble/Behance quality, 4K resolution
```

## Tips for Best Results

### Midjourney
- Add `--ar 16:9` for dashboard screenshots
- Add `--ar 4:3` for detail shots
- Use `--v 6` for latest version
- Add `--s 250` for stylized, `--s 50` for realistic

### DALL-E 3
- Be very specific about layout and composition
- Mention "UI mockup" or "Figma style" for interfaces
- Request "clean, professional, minimal" for SaaS aesthetic

### Stable Diffusion
- Use ControlNet for layout consistency
- Start with SDXL for UI mockups
- Use "ui design, figma, dribbble" in positive prompt
- Negative: "blurry, low quality, text artifacts, watermark"
